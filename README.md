# 🛕 Sri Ramalayam Temple - Doddavaram Construction Budget Tracker

Professional expense tracker for **Doddavaram Ramalayam Temple** construction.

## ✨ Features
- PIN-protected Editor access
- Real-time budget tracking with progress bars
- Receipt photo upload
- Mobile-first responsive design
- Export to PDF, Excel
- Data backup & restore

## 🚀 Quick Start
```bash
git clone https://github.com/seshagirich/doddavaram_ramalayam-temple.git
cd doddavaram_ramalayam-temple
npm install
npm run dev
```
